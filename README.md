📘**Site Escolar–README**
🖥️ **Sobre o Projeto**
Este é um site desenvolvido com HTML, CSS e JavaScript.
O objetivo é oferecer uma interface clara e moderna com as habilidades que tenho ate então.

**O projeto inclui:**
🌗 Modo claro e modo escuro

🎛️ Interatividade por botões

🔗 Links de redirecionamento interno e externo

🏫 Design voltado para uso escolar


✨**Funcionalidade**

🌗 Modo Claro/Escuro


🎛️ **Interatividade**

Elementos dinâmicos que respondem a ações do usuário.

🔗 **Links de Redirecionamento**

Navegação interna entre páginas do site.
Links seguros para plataformas externas (ex.: portal da escola, biblioteca digital etc.).

🎨 **Design Escolar**

Layout limpo com foco em usabilidade.
Cores suaves inspiradas no ambiente escolar.
Estrutura organizada para facilitar acesso rápido às informações.


🛠️ **Tecnologias Utilizadas**

🔹 HTML
°Estrutura do site

🔹 CSS
°Estilo, responsividade e animações

🔹 JavaScript
°Interatividade, lógica do tema e eventos


▶️**Como Executar o Projeto**
Abra o arquivo index.html no navegador de sua preferência.
Não é necessário instalar nenhum programa adicional — o projeto funciona 100% no front‑end.
